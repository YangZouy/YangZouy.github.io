github新建仓库并push本地代码过程

发表于2023-08-10|更新于2026-05-13|工具
|阅读量:

一、初始化

在项目的根目录下git bash

git init

二、github上新建仓库

直接new一个新的仓库 添加url以及描述 不添加readme

获取远程仓库的链接地址：

https://github.com/YangZouy/library.git

三、链接本地项目（仓库）与远程仓库

origin表示你为远程仓库起的名字，一般都叫origin 也可以是其它的，后面的链接就是远程仓库的真实地址

git remote add origin https://github.com/YangZouy/library.git

四、拉取github远程仓库主分支上的文件

现在一般主分支都是main

git pull origin main

五、代码提交并推送

// .表示将文件夹下所有文件都上传上去
git add .
git add *(表示文件/文件夹名)
// 提交到本地仓库
git commit -m "init"
// 推送到远程仓库origin 的main分支上
// git push <远程主机名><本地分支名>:<远程分支名>
git push -u origin master:main
// 查看当前状态
git status
